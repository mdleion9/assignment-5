(1)What is the difference between var, let, and const?

var: Function scoped and can be redeclared and updated. It is the old way of declaring variables in JavaScript.

let: Block scoped and can be updated but not redeclared in the same scope.

const: Block scoped and cannot be updated or redeclared after declaration.

(2)What is the spread operator (...)?

The spread operator (...) is used to expand elements of an array or object into another array or object.

(3) What is the difference between map(), filter(), and forEach()?

map(): Creates a new array by transforming each element.

filter(): Creates a new array with elements that match a condition.

forEach(): Loops through elements but does not return a new array.

(4) What is an arrow function?

An arrow function is a shorter way to write a function in JavaScript using =>.

(5)
What are template literals?

Template literals are strings written using backticks ` ` that allow embedding variables and expressions using ${}.
